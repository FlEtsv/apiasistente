Configuracion pendiente

Por que esta configuracion es critica

Si la URL de Ollama o un modelo estan mal, el chat puede fallar, tardar demasiado o devolver respuestas vacias.

El modelo de embeddings alimenta el retrieval del RAG: si no es correcto, el sistema no encuentra contexto y baja la precision.

El scraper es opcional: sin URLs validas no ingesta conocimiento web y la base RAG no crece con informacion externa.

El preset inicial viene con nuestra configuracion recomendada, pero el cliente final puede cambiar todos los campos.

Robot RAG y Scraper

Robot RAG: revisa el corpus en segundo plano, detecta contenido degradado y mantiene la base util para retrieval.

Scraper: extrae texto de webs que indiques y lo inserta como documentos del RAG.
